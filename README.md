# 📸 AI Photo Editor (OpenCV + Streamlit)

An interactive Photo Editing web application built using OpenCV and Streamlit. This project allows users to upload images, apply real-time filters, and download edited results.

---

## 🚀 Features

- 📤 Upload images (JPG, PNG, JPEG)
- 📏 Resize images dynamically
- 🎚 Adjust brightness & contrast
- 🎨 Apply filters:
  - Grayscale
  - Blur
  - Sharpen
  - Warm filter
- 🧑‍🎨 Portrait-style background blur
- ✏ Edge detection
- 🎭 Cartoon effect
- 📥 Download edited images

---

## 🧠 Key Learnings

- Image processing with OpenCV
- Real-time UI development using Streamlit
- Handling RGB vs BGR image formats
- Building end-to-end Computer Vision applications

---
